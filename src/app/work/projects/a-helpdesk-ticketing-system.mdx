## Overview

Helpdesk, is a great tool if you want to manage your projects, tickets and be here for your clients It comes also with more than 60 languages ready to use. All this made with the best technologies.

## Key Features

- **Component Library**: Built a set of modular, reusable UI components using Livewire and styled-components in Laravel, focusing on accessibility and responsiveness.
- **Theming and Customization**: Integrated a theming system that allows easy switching and customization of color palettes, typography, and layout styles using CSS variables and Figma tokens.
- **Documentation and Usage Guidelines**: Developed comprehensive documentation with Storybook to showcase components, usage patterns, and best practices, ensuring the design system is easy to adopt by end-users.

## Technologies Used

- **Laravel**
- **Filament**
- **Livewire**
- **MySQL**
- **Pusher account**

## Outcome

The design system is now actively used across multiple projects, leading to faster development cycles, fewer design inconsistencies, and conducted root cause analysis to resolve recurring technical issues, reducing ticket volume by 70%.
